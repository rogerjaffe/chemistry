Calculating K

This page permits you to work on calculating the value of K. When you press "New Question", a window with a balanced equation and a set of equilibrium concentrations will open. Calculate the value of K, enter it in the cell and press "Check Answer". Results appear in the table on the main page. If the small window covers up any part of the main window, it can be dragged to a new position. Before beginning, enter your name, select your instructor and press "Submit."
