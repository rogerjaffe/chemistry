Simple Mixtures

This is an introductory page to dealing with simple mixtures. The problems posed will involve mixtures consisting of two elements which are solids. You will need a periodic table or a table of the elements which includes atomic weights and densities. Again, since this is an introductory treatment, you need not be concerned about whether the mixture is a simple hterogeneous one, or results in an alloy. Just focus on using the data provided to solve the problem. When you press "New Problem", a question will appear in a new window. Calculate the answer and enter it in the cell in the new window. Then press "Check Answer". The results appear in the table on the main window.
