Calculating Eo

This page provides basic exercises for calculating standard potentials. When you press "New Problem", an unbalanced redox reaction with its Eo will appear. Calculate the value of Eo, enter it in the answer cell and press "Check Answer." Results will appear in the main table. The questions will provide a redox reaction. You do not have to balance it completely, but you do need to determine the number of electrons transferred in the balanced equation.
  • Be careful about the sign. Positive values can be entered as is (do not use a "+"+ sign. Negative values must have a "-" sign.
  • If you get a problem "incorrect", you should redo it and recheck your answer.
  • If you miss a problem three times, pressing the "Show Answer" button will cause a complete solution to appear.
