Balancing Chemical Reactions

This page is an exercise in balancing chemical reactions. When you press "New Reaction," a reaction will appear in the top frame on this page. In front of each chemical species will be an empty cell. Put the proper coefficient in each cell and then press "Check Answer". The results appear in the table. Hints:
  • Use only the smallest set of integer coefficients.
  • Use TAB to move between cells.
