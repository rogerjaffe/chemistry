Make a Better Guess #1

This page presents random "word" problems in the area of elements, mass and moles which are to be done without a calculator. You are to estimate the answer to one significant figure using only the periodic table and mental calculations. If you write anything down or use a calculator, you are defeating the purpose of the exercise. When you press "New Question", a question will appear in the top frame. Estimate the answer and enter it in the cell. Then, press "Check Answer" and the results appear in the table. If you get a problem wrong, you should redo it until you get it correct, but you can press "New Question" at any time. You need a periodic table or chart of the elements to do these problems.
